Core API¶

GET (?i)/(?P<exp_id>[0-9]+)/getaction.json¶

Get an action given a context for a specific exp_id

Example
http://example.com/1/getAction.json?key=XXXX&context={‘age’: 25}
Parameters:
  • exp_id (int) – Experiment ID as specified in the url
  • context (JSON) – The context to be evaluated.
  • key (string) – The key corresponding to the experiment.
Returns:

A JSON of the form: {“action”: XX}
Raises AuthErorr:
 

401 Invalid key
GET (?i)/(?P<exp_id>[[0-9]+)/setreward.json¶

Update the parameters for a given experiment

Example
http://example.com/1/setReward.json?key=XXXX&context={‘age’: 25} &action={‘action’:’A’}&reward={‘click’:1}
Parameters:
  • exp_id (int) – Experiment ID as specified in the url
  • context (JSON) – The context to train on.
  • action (JSON) – The action to train on.
  • reward (JSON) – The reward for the experiment.
  • key (string) – The key corresponding to the experiment.
Returns:

A JSON of the form: {“status”:true}
Raises KeyError:
 

400 Error if Key is not valid.
