Statistics API¶

GET (?i)/stats/(?P<exp_id>[0-9]+)/getcurrenttheta.json¶

Get the current theta for experiment id exp_id

Example
http://example.com/stats/1/getCurrentTheta.json
Requires:

A secure cookie, obtained by logging in.
Parameters:
  • exp_id (int) – The experiment ID for the theta that is to be retrieved.
Returns:

A JSON of current theta.
Raises:

AUTH_ERROR if there is no secure cookie available.
GET (?i)/stats/(?P<exp_id>[0-9]+)/gethourlytheta.json¶

Get a dict of all logged thetas

Example
http://example.com/stats/1/getHourlyTheta.json
Requires:

A secure cookie, obtained by logging in.
Parameters:
  • exp_id (int) – The experiment ID for the thetas that are to be retrieved.
Returns:

A JSON of JSONs of hourly logged thetas.
Raises:

AUTH_ERROR if there is no secure cookie available.
