Admin API¶

POST (?i)/admin/exp/add.json¶

Create a new experiment

Requires:

A secure cookie obtained by logging in.
Parameters:
  • name (string) – Name of the experiment
  • getaction (string) – String of python code for get action code
  • setreward (string) – String of python code for set reward code
  • adviceid (bool) – Bool indicating whether adviceIds are used
  • hourly (bool) – Bool indicating whether the state of Theta should be stored hourly.
Returns:

A JSON of the form: { id : the assigned experiment id,

name : the name of the experiment (checked for duplicates), error : (optional) error message }
Raises AUTH_ERROR:
 

If no secure cookie available.
GET (?i)/admin/exp/list.json¶

Retrieve a list of experiments running on this server

Example
http://example.com/admin/exp/list.json?max=10
Requires:

A secure cookie obtained by logging in.
Parameters:
  • max (int) – Indicating the max length of the list.
Returns:

A JSON containing a list of {expid : name} pairs
Raises AUTH_ERROR:
 

If not secure cookie available.
